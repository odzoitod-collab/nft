# Руководство для воркера: сайт и панель воркера

Краткий туториал по работе с **сайтом Ethos Gallery** (Mini App) и **панелью воркера** в Telegram-боте.

---

## 1. Кто такой воркер и что такое рефералы

**Воркер** — пользователь бота, который привлекает клиентов по своей реферальной ссылке. Каждый, кто перешёл по этой ссылке и нажал «Старт» в боте, становится **рефералом** этого воркера и привязывается к нему в системе (в БД заполняется `referrer_id`).

- Рефералы видят на сайте ваши персональные цены на NFT (если вы их задали).
- Для рефералов могут действовать ваши лимиты на пополнение и вывод (если вы их задали).
- Вы видите список рефералов, можете менять им баланс, верификацию, цены NFT и писать им сообщения от имени биржи.

---

## 2. Сайт (Mini App) — как им пользуется клиент

Пользователь открывает приложение по кнопке **«Открыть приложение»** в боте или через Menu Button. На сайте доступно:

| Раздел | Описание |
|--------|----------|
| **Маркет** | Каталог NFT: покупка и продажа. Цены берутся из каталога; у реферала — персональные цены воркера, если заданы. |
| **Портфель** | Вкладка «Портфель» — NFT, которые есть у пользователя. Вкладка «Проданые» — история проданных NFT. |
| **Сезон** | Сезонный конкурс, призы за объём торгов. |
| **Профиль** | Баланс, история операций, настройки, пополнение и вывод. |

**Пополнение и вывод:** пользователь выбирает страну (RUB, UAH, PLN, KZT, EUR), вводит сумму. Система проверяет **минимальный депозит** и **минимальный вывод**: для реферала используются лимиты его воркера (если заданы), иначе — глобальные лимиты из админки.

## 3. Панель воркера в боте (`/worker`)

В боте отправьте команду **`/worker`**. Откроется меню с реферальной ссылкой и кнопками.

### 3.1 Реферальная ссылка

В сообщении показана ваша ссылка вида:
`https://t.me/BOT_USERNAME?start=ВАШ_КОД`

Делитесь этой ссылкой. Кто перейдёт и нажмёт «Старт», станет вашим рефералом.

### 3.2 Мои рефералы

- Нажмите **«👥 Мои рефералы»** — откроется список рефералов (имена/ники).
- Нажмите на имя — откроется **профиль реферала**:
  - **💰 Изменить баланс** — ввести новый баланс в TON (сохраняется в БД).
  - **✅ Дать вериф** / **❌ Забрать вериф** — выдать или снять верификацию.
  - **💰 Цена NFT для реферала** — задать персональную цену на конкретный NFT только для этого реферала (см. ниже).
  - **📩 Написать в бот** — ввести текст; сообщение придёт рефералу в бот как «Сообщение от биржи».

### 3.3 Добавить клиента по нику/ID

Если человек уже пользуется ботом, но не заходил по вашей ссылке, его можно привязать вручную:

- Нажмите **«➕ Добавить клиента по нику/ID»**.
- Введите **@username** (например `@durov`) или **Telegram ID** пользователя.
- Пользователь должен хотя бы раз открыть бота. После привязки он станет вашим рефералом. Уведомление ему не отправляется.

## 4. Как изменить цену NFT для реферала

Цена в каталоге на сайте — общая для всех. Для **конкретного реферала** вы можете задать свою цену на конкретный NFT — тогда на маркете он будет видеть вашу цену, остальные — из каталога.

**Шаги:**

1. `/worker` → **«👥 Мои рефералы»** → выберите реферала.
2. В профиле реферала нажмите **«💰 Цена NFT для реферала»**.
3. Введите **код NFT** из каталога (тот же, что в Supabase в таблице `nft_catalog`, поле `code`).
4. Бот покажет текущую цену в каталоге и (если уже задана) цену для этого реферала. Введите новую цену в **TON**.
5. Готово: на сайте этот реферал при открытии маркета будет видеть указанную цену по данному NFT.

Чтобы задать цены по нескольким NFT — повторите шаги 2–4 для каждого кода.

## 5. Дуо-токены: что это и как создать

**Дуо-токен** — это NFT, для которого действует правило «пара для продажи»:

- **Купить** можно один такой NFT (как обычно).
- **Продать** можно только если у пользователя есть **два** таких NFT (пара). Цена в листинге указывается за пару.

Так можно делать, например, парные коллекционные предметы или механики «собери пару».

**Как сделать NFT дуо-токеном:**

1. NFT уже должен быть в каталоге (таблица `nft_catalog` в Supabase). Если его нет — сначала добавляет админ через админку.
2. `/worker` → **«🔄 Создать дуо-токен»**.
3. Введите **код NFT** (как в каталоге).
4. Бот пометит этот NFT как дуо. Реферал сможет купить один такой NFT; для продажи нужна пара (2 шт).

Отменить дуо через бота нельзя — только через БД (поле `is_duo` в `nft_catalog`).

## 6. Минимальный депозит и минимальный вывод для рефералов

Вы можете задать свои лимиты **только для своих рефералов**:

- **Минимальный депозит** — ниже этой суммы реферал не сможет отправить заявку на пополнение.
- **Минимальный вывод** — ниже этой суммы реферал не сможет отправить заявку на вывод.

Если лимиты не заданы (или сброшены), у реферала действуют **глобальные** лимиты из админ-панели.

**Как задать:**

1. `/worker` → **«📥 Мин. депозит для ваших рефералов»** или **«📤 Мин. вывод для ваших рефералов»**.
2. Бот покажет текущее значение (или «глобальный»).
3. Отправьте **число** (сумма в TON), например `10` или `50`.
4. Чтобы снова использовать глобальный лимит — отправьте **0**.

Значения сохраняются в настройках (`worker_ВАШ_TELEGRAM_ID_min_deposit_ton` и `worker_ВАШ_TELEGRAM_ID_min_withdraw_ton`) и подставляются на сайте при пополнении/выводе реферала.

## 7. Краткая шпаргалка

| Действие | Где |
|----------|-----|
| Реферальная ссылка | `/worker` — в тексте сообщения |
| Список рефералов | `/worker` → Мои рефералы |
| Баланс / вериф / цены NFT / сообщение рефералу | Профиль реферала (Мои рефералы → имя) |
| Цена NFT для реферала | Профиль реферала → «Цена NFT для реферала» → код NFT → цена в TON |
| Сделать NFT дуо-токеном | `/worker` → Создать дуо-токен → код NFT |
| Мин. депозит для рефералов | `/worker` → Мин. депозит для ваших рефералов → число или 0 |
| Мин. вывод для рефералов | `/worker` → Мин. вывод для ваших рефералов → число или 0 |
| Добавить клиента без ссылки | `/worker` → Добавить клиента по нику/ID → @ник или ID |

Глобальные лимиты (для всех, у кого нет воркера) и каталог NFT настраиваются в **админ-панели** (`/admin`), доступной только администраторам.
